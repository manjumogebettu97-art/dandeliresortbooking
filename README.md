# Dandeli Resorts Booking

A single-page booking website for **Dandeli Resorts Booking** — resorts, cottages,
homestays and camping packages in Dandeli, Karnataka, on the banks of the river Kali.

## Features

- Stay packages from ₹1199/person (camping, homestay, cottages, inn, premium cottage)
- "What's included" — 3 meals, river activities, trekking and in-house entertainment
- Sightseeing, booking steps, photo gallery with lightbox, and FAQ
- Enquiry form that opens a pre-filled WhatsApp message (no backend required)
- Floating WhatsApp button and click-to-book CTAs throughout
- Fully responsive with scroll-driven animations and a premium champagne-gold theme

## Tech

Plain static site — no build step:

- `index.html` — markup
- `style.css` — styles (Cormorant Garamond + Jost, champagne-gold palette)
- `script.js` — nav, scroll effects, animated counters, gallery lightbox, WhatsApp form
- `images/` — optimized photos

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
